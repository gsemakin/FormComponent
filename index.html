    Please check your Divisional SMP before building a lead generating form to ensure the request matches the lead behaviour. The SMP may have been reviewed since this example form was built.

    The TMC SMP can be accessed from the EMEA DO Sharepoint Site

    Notes to developers

    MQL form fields are hidden and NOT mandatory, unless the salesRequest field is checked.

    For switching between without/with salesRequest field use a method form1.settings.leadGenType = "TMC_Basic";
    or
    form1.settings.leadGenType = "TMC_CA";

    Text field Preferred Distributor field is only for Sample requests (appears when №of operators >= 6).

    If you need to change Preferred Distributor from a 'text' type (by default) to a SELECT type please add:
    form1.changeFieldType('selDist');
    and then add relevant options to it (use method setOptions).

    If Company Type is not 'End user', leadGen fields will not appear.

    Business Phone should only appear if "Mobile Phone" has not been filled.
